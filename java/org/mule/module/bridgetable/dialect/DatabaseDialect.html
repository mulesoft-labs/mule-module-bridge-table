public abstract String getCreateTableSQL (String tableName)

Returns the DLL statement to create the bridge table taking into account the table name and the column types. This script should create indexes if required and should ensure that both key1 and key2 are unique. This means that all relations are 1 to 1 and there cannot be two different values for key2 associated with the same key1 (and viceversa)

tableName The name of the table
