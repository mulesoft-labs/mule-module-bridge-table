BridgeTable

org.mule.module.bridgetable.BridgeTable
Known Indirect Subclasses
DataSourceBridgeTable, InMemoryBridgeTable
DataSourceBridgeTable Concrete implementation of a bridge table that uses a JDBC compliant database to store key mappings. 
InMemoryBridgeTable Concrete implementation of a bridge table that uses two maps in memory to store key mappings. 

Class Overview

A bridge table maps key1 to key2, allowing to do direct and reverse lookups. This mean that knowing key1 you can get the corresponding value for key2 and viceversa. This mapping is 1-1, so just one value of key1 and one value of key2 are allowed.
